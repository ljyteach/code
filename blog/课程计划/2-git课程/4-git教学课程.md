### 五、文件状态 常用命令 参数设置
#### 状态：工作区、版本库（暂存区、分支）
#### 常用命令
+ 创建版本库     git init
+ 添加到仓库     git add . (或文件名)  将工作区变动添加到暂存区
+ 提交到仓库     git commit -m "提交说明";  将暂存区内容提交到当前分支
+ 查看状态       git status
+ 查看变动       git diff
+ 查看提交日志   git log
+ 回溯版本       git reset --hard
+ 撤销修改       git checkout  --文件名
+ 删除文件       git rm
+ 添加远程库     git push
+ 获取远程库     git pull
+ 创建分支       git branch 分支名称
+ 切换分支       git checkout 分支名称
+ 合并分支       git merge
+ 查看远程分支   git remote -v     -v 查看详细信息
+ 创建标签       git tag
